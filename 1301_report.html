台塑 2023-2024 Q3 營運報告差異分析

營收結構變化

✓
整體營收比較：2024年第三季營收約504.92億台幣，相較2023年第三季的507.36億台幣略降0.5%。此數據可在報告第5頁營收數據中得到驗證。
✓
PE類產品(HDPE/EVA/LLDPE)營收占比從2023年Q3的24%下滑至2024年Q3的19%，顯示PE產品線營收貢獻明顯減少。此變化可在第2頁圓餅圖中清楚觀察到。
✓
特用化學品(AN/MMA/ECH/MTBE)的營收占比從9%微幅上升至10%，表現相對穩定。

獲利表現

✗
原分析需要更正：2023年第三季的主要獲利來源描述不夠精確。報告中並未明確指出EVA為主要獲利來源。
✓
2024年第三季的主要獲利來源為PVC和液碱(Caustic Soda)，這點在報告標題中有明確說明。

稅前利益走勢

✓
2023年第三季稅前利益達68.38億台幣（6,838百萬元）。
✓
2024年同期轉為虧損32.45億台幣（-3,245百萬元）。
✓
業績下滑主要原因：
  • 營業收入的減少
  • 來自台塑石化(FPCC)、美國子公司(FPC-USA)以及福建福欣特殊鋼鐵的投資收益減少

產能擴充計畫調整

✓
2023年報告中包含寧波600KMT的丙烯產能擴充計畫，投資金額為8億美元。
✓
2024年報告中已不見寧波丙烯產能擴充計畫，轉而更聚焦於台灣本地的PVC產能擴充計畫，顯示公司在產能布局策略上有所調整。
